<h1>FitTrack - Predictive Modelling of Daily Calorie Requirements using User Physiological Data</h1>

<p>
FitTrack is a <b>machine learning powered health analytics web application</b> that predicts 
<b>exercise calorie burn and daily calorie requirements</b> using physiological data such as 
age, weight, height, heart rate, exercise duration, and body temperature.
</p>

<p>
The application is built using <b>Python and Streamlit</b> and utilizes an 
<b>ensemble machine learning model combining XGBoost, LightGBM, and CatBoost</b> to generate 
accurate calorie burn predictions.
</p>

<p>
For model transparency and interpretability, the application integrates 
<b>SHAP (SHapley Additive Explanations)</b>, allowing users to visualize and understand 
which physiological factors most influence calorie burn predictions.
</p>

<p>
Additionally, FitTrack includes a <b>nutrition logging system</b> based on an 
<b>Indian food nutrition dataset</b>, enabling users to track daily intake of 
<b>calories, protein, carbohydrates, and fat</b>.
</p>

<p>
Interactive dashboards display <b>daily calorie progress, remaining calorie targets, 
and macro nutrient distribution</b>, helping users effectively monitor their 
diet and exercise impact.
</p>

<p>
Overall, FitTrack provides a <b>comprehensive and interactive platform</b> for users to 
track physical activity, manage nutrition intake, and gain insights into the 
physiological factors influencing their calorie expenditure.
</p>
